# MERN Blog Application

A full-stack blogging platform built with the MERN stack.

## Features

- User Authentication
- JWT Cookie Authentication
- Role-Based Authorization
- Create, Read and Delete Blogs
- Image Uploads
- Comment System
- Responsive UI
- Error Handling
- Loading States

## Tech Stack

Frontend

- React
- React Router
- Bootstrap

Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- Multer
- Cloudinary
- JWT
