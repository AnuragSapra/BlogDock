# BlogDock

A full-stack blogging platform built with the MERN stack.

## Features

- User Authentication
- JWT Cookie Authentication
- Role-Based Authorization (Admin/User)
- Create Blogs
- View Blogs
- Delete Blogs (Admin Only)
- Image Uploads with Cloudinary
- Comment System
- Responsive Design
- Loading & Error States

## Tech Stack

### Frontend

- React
- React Router
- Bootstrap
- Vite

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- Multer
- Cloudinary
- JWT
